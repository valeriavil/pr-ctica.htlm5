Comenzar

¿Qué es HTML5?

HTML5 (HyperText Markup Language versión 5) es el lenguaje de marcado estándar para crear y estructurar contenido en la web. Sustituyó a versiones anteriores en 2014, incorporando nuevas funcionalidades nativas para multimedia, formularios mejorados y etiquetas semánticas. Sirve para definir la estructura básica de una página web (títulos, párrafos, imágenes, enlaces, etc.) y es compatible con todos los navegadores modernos, siendo fundamental para el desarrollo web ya que permite que el contenido sea accesible y bien organizado para usuarios y motores de búsqueda.

Estructura básica de HTML5

<!DOCTYPE html> <html> <head> <title>Mi página</title> </head> <body> </body> </html>

- <!DOCTYPE html>: Declaración que indica al navegador que se trata de un documento HTML5, asegurando la correcta interpretación del código.
- <html>: Etiqueta raíz que engloba todo el contenido del documento, es el contenedor principal.
- <head>: Sección que contiene metadatos (información sobre el documento), como título, codificación de caracteres, enlaces a hojas de estilo o scripts.
- <body>: Sección que incluye todo el contenido visible para el usuario (textos, imágenes, enlaces, formularios, etc.).

Etiquetas principales

h1 - h6

Etiquetas para títulos y subtítulos, donde h1 es el más importante (título principal) y h6 el menos relevante. Ayudan a estructurar el contenido y mejorar la accesibilidad y posicionamiento web.

p

Define párrafos de texto, separando bloques de contenido textual para mejorar la legibilidad y organización del documento.

img

Inserta imágenes en la página. Requiere el atributo "src" (ruta de la imagen) y "alt" (texto alternativo para accesibilidad y cuando la imagen no carga).

a

Crea enlaces (hipervínculos) a otras páginas web, archivos o secciones dentro del mismo documento. El atributo "href" define la dirección de destino.

button

Crea botones interactivos que pueden ejecutar acciones (como enviar formularios o activar scripts) mediante eventos de JavaScript.

ul / ol / li

ul: Crea listas desordenadas (con viñetas). ol: Crea listas ordenadas (con números o letras). li: Define cada elemento dentro de ambas listas.

HTML5 semántico

header

Define la sección de encabezado de un documento o sección, generalmente contiene logotipos, títulos, navegación o información de contacto.

nav

Contiene los enlaces de navegación principales del sitio web, agrupando rutas para facilitar la navegación del usuario.

main

Representa el contenido principal y único del documento, excluyendo elementos repetitivos como encabezados, pies de página o navegación secundaria.

section

Agrupa contenido relacionado que forma una sección lógica del documento (ej: una sección de noticias, una sección de características).

article

Define contenido independiente y autocontenido que podría ser distribuido o reutilizado por sí mismo (ej: un artículo de blog, una entrada de foro).

footer

Representa el pie de página de un documento o sección, generalmente contiene información de copyright, enlaces de contacto, políticas de privacidad o enlaces a redes sociales.

Formularios en HTML5

Los formularios en HTML5 permiten recopilar datos del usuario y enviarlos a un servidor para su procesamiento. Incluyen etiquetas y elementos específicos: - form: Contenedor principal del formulario, define la acción (ruta del servidor) y método de envío de datos (GET o POST).
- label: Etiqueta descriptiva para elementos de formulario, mejora la accesibilidad y permite seleccionar el campo al hacer clic en el texto.
- input: Elemento versátil que puede crear campos de texto, correos electrónicos, contraseñas, botones de opción, casillas de verificación, etc., según su atributo "type".
- textarea: Campo de texto multilínea para comentarios o textos extensos.
- select: Crea listas desplegables de opciones, con elementos "option" dentro.
- button: Botón para enviar el formulario o ejecutar acciones.

Multimedia en HTML5

HTML5 incorporó soporte nativo para multimedia, eliminando la necesidad de plugins externos (como Flash). Permite integrar imágenes, audio y video de forma sencilla.

Imágenes

La etiqueta img se usa para insertar imágenes; sus atributos principales son src (ruta), alt (texto alternativo), width y height (dimensiones). También existen formatos optimizados como WebP para mejorar el rendimiento.

Audio

La etiqueta audio permite reproducir archivos de sonido. Soporta formatos como MP3, WAV y OGG. Incluye atributos como controls (muestra controles de reproducción), autoplay (reproduce automáticamente) y loop (reproduce en bucle).

Video

La etiqueta video inserta videos en la página, soportando formatos como MP4, WebM y OGG. Atributos destacados: controls, autoplay, loop, width/height, y poster (imagen que se muestra antes de reproducir).

Accesibilidad

El atributo alt en imágenes es crucial para accesibilidad: describe el contenido de la imagen a usuarios con lectores de pantalla y se muestra si la imagen no carga. Para audio y video, se recomienda agregar subtítulos o transcripciones.

Conclusión

HTML5 es una herramienta fundamental en el desarrollo web moderno, ya que no solo estructura el contenido de forma clara, sino que también incorpora funcionalidades nativas que mejoran la experiencia del usuario, la accesibilidad y la compatibilidad entre dispositivos. Su enfoque semántico ayuda a organizar el contenido de manera que sea entendible tanto para usuarios como para motores de búsqueda, mientras que su soporte para multimedia y formularios optimiza el desarrollo sin depender de plugins externos. Sin HTML5, la web tal como la conocemos hoy no existiría en su forma actual, y seguirá siendo la base sobre la que se construyen aplicaciones y sitios web innovadores.
